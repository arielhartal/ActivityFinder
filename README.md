# ActivityFinder
ActivityFinder is an Android app developed as part of a school project, aimed at helping individuals connect with others who share a similar interest in specific activities. Whether you're looking for a workout partner, a group for a hiking trip, or someone to join you for a game of basketball, ActivityFinder makes it easy to discover people nearby who want to participate in the same activity.

The app was primarily built in Java, integrating a robust database to store user profiles, activities, and preferences. Additionally, it leverages the Google Maps API to provide location-based suggestions, allowing users to find others close to their location or preferred meeting spots. The app offers an intuitive user interface where users can browse activities, set filters, and reach out to potential companions.

With features such as real-time updates, notifications for new matches, and detailed activity categories, ActivityFinder simplifies the process of finding like-minded individuals to engage in activities together.
